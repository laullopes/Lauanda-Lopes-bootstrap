Element

Element används i HTML för att ge struktur till hemsidan. De brukar består av en start och sluttag.

HTML

Det står för Hypertext Markup Language och det är ett märkspråk som används för att skriva hemsidor.

ID

Det är ett sätt att namna information på hemsidan så att man kan formatera på olika sätt samt länka till bestämda områden på en hemsidan. ID ska vara unikt på en hemsidan och man brukar använda det på till ex. sidhuvud, sidomenyer, sidfot, osv.

Class

Class fungerar nästan som ID, skillnaden är att man kan använda samma klass till flera element och man kan inte direktlänka till en klass.

Meta

Meta är ett tag som man använder för att ge en beskrivning på hemsidan och denna beskrivningen kommer ofta upp i sökresultaten.

Responsive

De flesta hemsidor idag är responsiva vilket betyder att de anpassar med storleken på skärmen. Till exempel om du använder en Ipad och sedan jämför på hur hemsidan ser ut på din mobils så kommer de ser likadan ut eller kanske de har ett helt annat stil.

Http

Det står för Hypertext Transfer Protocol, det är en typ av nätverkskommunikation som används för att ladda hemsidor. Enligt detta protokoll kan maskiner ta emot och skicka textinnehåll, koder som leder till att sidan öppnas av webbläsaren.

Https

Hypertext Transfer Protocol Secure, det har ett skyddslager i dataöverföringen mellan datorn och servern. På webbplatser med en HTTPS-adress krypteras kommunikation vilket ökar datasäkerheten avsevärt.

Compressed data

Datakomprimering krymper stora filer till mycket mindre. Det gör det genom att bli av med onödiga data samtidigt som informationen i filen sparas.

CDN

Det står för Content Delivery Network och det är en nätverk av servrar som lagrar repliker av innehållet på andra webbplatser i minnet cache och sedan levererar dem till besökare baserat på den geografiska platsen. För att då anslutar dem till snabbaste och närmaste server, vilket minskar dataöverförings tid.

Local bootstrap

Det använder jag i denna uppgiften.

JQuery

Det är ett Javascript-bibliotek som underlättar livet för dem som använder Javascript, med det kommer man kunna göra mycket mer med mindre.

Flexbox

Flexbox dynamiskt organisera elementen på en HTML-sida i dess behållare. Det vill säga, oavsett deras dimensioner, kommer de alltid att ha en flexibel layout inom sitt överordnade element, omorganisera och efter behov.

Grid

Som flexbox, det är också ett sätt att skapa layouter. Men Grid erbjuder fler fördelar för att det är lättare att förstår och läsa koden eller att den är responsiv i alla webbläsare.

Code snippets

Det är kodblock som används i Visual Studio för att effektivisera kod utveckling.
